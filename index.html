Roboin Humanoid Plus Project (RH+)

This project explores fundamental robotics concepts by building a 2-axis manipulator,
a 6-axis manipulator, and a humanoid robot. Our main focus is on mechanical structures,
environment perception, and system control using ROS2 and other tools.

2-Axis Manipulator

Notion Page GitHub Code

Explanation

The 2-Axis Manipulator is designed for planar motion in an X-Y plane. By controlling two servos, we can move the end-effector to draw shapes or reach specified coordinates with precision.

Project Overview

Major goals include coordinate-based motion control via ROS2, shape-drawing on a flat surface, and camera integration for basic color or marker detection. This serves as a simpler introduction before moving on to higher degrees of freedom.

Additional Commentary

With fewer joints to manage, the 2-axis system allows rapid prototyping and straightforward ROS2 integration. We plan to enhance its capabilities by adding real-time vision processing and more efficient trajectory planning in future iterations.

6-Axis Manipulator

Notion Page GitHub Code

Explanation

The 6-Axis Manipulator provides significantly more flexibility and movement options. It can approach objects from multiple angles, handle complex pick-and-place tasks, and perform advanced operations such as obstacle avoidance.

Project Overview

Our objectives include using MoveIt2 for path planning in a 3D workspace, leveraging depth cameras for environmental awareness, and performing multi-step tasks like stacking or sorting various items.

Additional Commentary

Developing a 6-axis arm involves a more complex mechanical design and advanced kinematic configurations. By integrating ROS2 and MoveIt2, we aim to achieve precise real-time control. Future enhancements will involve better object recognition and grasping algorithms to handle diverse tasks.
